# KoalaFlyff — Multi-Boxing Helper

A lightweight browser extension designed for **Flyff Universe** that enables streamlined multi-boxing by routing keystrokes between browser tabs.

## 🚀 Features
- **Cross-Tab Key Mapping**: Listen for keys on one tab and execute them on another.
- **Multiple Mappings**: Configure up to 10 unique key pairs (e.g., press `C` on Main to cast `Heal` on Support).
- **Service Toggle Hotkey**: Toggle the entire service on/off instantly using a global hotkey (Default: `Ö`).
- **Session History**: Track your playtime and session durations automatically.
- **Deep Integration**: Specifically optimized for WebGL games like Flyff Universe.
- **Privacy First**: Minimal permissions, no external data tracking, and entirely private local storage.

## 🛠 Usage
1. Open Flyff Universe in two separate browser tabs.
2. Open the **KoalaFlyff** popup.
3. Select your **Main Character** (Listener Tab) and your **Support Character** (Target Tab).
4. Add your desired key mappings.
5. Click **Start Service** or use the hotkey (`Ö`) to begin.

## ⚙️ Configuration
- **Hotkey**: Change the toggle key in the Settings panel (next to the logs).
- **Logs**: Enable debug logging if you want to see detailed routing information.
